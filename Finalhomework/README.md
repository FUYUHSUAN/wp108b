## 前言:
* 我的作業從5/28就一步一步起程了，一開始我確實沒什麼想法，原本想直接做筆記，但後來發現做一份主題式的應用網頁，更能在其中邊做邊學習到許多東西，因此我選擇了"點餐系統POS的應用"。
## 創作想法來源:
* 其實，我在做網頁時並不是一次就能想好整個架構圖的，我都是在忽然間有想法時就記錄下來，並將其應用於我的作業，所以整體來說，我的作業是經過長時間洗禮，慢慢想出來的
